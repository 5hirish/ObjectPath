Installing and running

Python

Install newest version from GitHub:

$ git clone https://github.com/adriank/ObjectPath.git
$ cd ObjectPath
$ python shell -h
usage: shell [-h] [-u URL] [-xml] [-d] [FILE]

Command line options

positional arguments:
	FILE               File name

optional arguments:
	-h, --help         show this help message and exit
	-u URL, --url URL  URL containing JSON document.
	-xml               [EXPERIMENTAL] Expect XML input.
	-d, --debug        Debbuging on/off.

If you want to use localization features you need python-tz package installed on your machine.

You can also run ObjectPath in PyPy, the faster Python implementation:

pypy objectpath
Heads up! For a limited set of ObjectPath functionality you can open developer console of your choice (right click here and click on inspect element and then on console) and use ObjectPath JavaScript implementation to test the language without installing it on your box. Just write: 
op.exe("2+2")

Basics

Basic arithmetics

ObjectPath supports common arithmetics:

  • addition (+)
  • subtraction (-)
  • multiplication (*)
  • division (/)
2+2*2 -> 6
(2+2)*2 -> 8

Working with strings

ObjectPath provides some tools to manipulate strings. Strings are written as:

"text" or 'text'

Strings can be concatenated by + sign:

"Hello "+"World!" -> "Hello World!"

Numbers can be concatenated with strings if string is on the left of + sign

"Hello "+2+"!" -> "Hello World!"
2+" Hello!" -> error (in JS you'll see NaN which is a bug)
""+2+" Hello!" -> "2 Hello!"
